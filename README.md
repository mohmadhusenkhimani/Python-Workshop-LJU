# ▶️ How to Run

Clone this repository:

```bash
git clone https://github.com/mohmadhusenkhimani/Python-Workshop-LJU.git
```

Move into the project folder:

```bash
cd Python-Workshop-LJU
```

Run any Python program:

```bash
python filename.py
```

Example:

```bash
python Q1_StringMethods.py
```
